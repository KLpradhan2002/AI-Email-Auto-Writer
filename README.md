# AI Email Reply Generator

## 🚀 Features
- Generate AI replies inside Gmail
- Chrome Extension integration
- Spring Boot backend
- React frontend

## 🛠 Tech Stack
- React (Vite)
- Spring Boot
- Gemini API
- Chrome Extension

## ▶️ Run Locally

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
mvn spring-boot:run

### Extension
Load unpacked from /extension in Chrome
